# Spring Core 핥아먹기

# 개요

> ***핥다[할따]
물, 불, 빛, 시선 따위가 물체의 표면을 부드럽게 스치거나 비추거나 머무르다.***
> 

Spring 6 릴리즈를 기념해, Spring 공식 문서를 읽으며 정리한 시리즈입니다.

Spring Core와 Spring Context에 대해 살펴봅니다.
